# 🌆 Jellyfin Cyberpunk Skin

> _"Welcome to the neon grid. This isn’t just a skin — it’s an interface upgrade from another timeline."_

---

## 🚀 Overview

Dieser Skin verwandelt dein Jellyfin in eine düstere, neon-getränkte Cyberpunk-Oberfläche. Inspiriert von futuristischen Megacities, holographischen Interfaces und digitalen Unterwelten bringt er Style, Atmosphäre und Edge direkt in dein Media-Setup.

## 🧬 Design Philosophy

> _"Form follows function — but style defines dominance."_

Dieser Skin ist nicht nur Optik — er ist ein Statement.  
Jede Linie, jedes Leuchten und jede Transparenz ist darauf ausgelegt, dein Jellyfin wie ein Terminal aus einer dystopischen Zukunft wirken zu lassen.

---

## 📦 Installation

```bash
git clone https://github.com/yourname/jellyfin-cyberpunk-skin.git
```

1. Dateien in dein Jellyfin Custom CSS Verzeichnis kopieren  
2. CSS in den Einstellungen aktivieren  
3. System rebooten oder Cache leeren  

---

## 🛠️ Customization

```css
:root {
  --primary-color: #00fff7;
  --secondary-color: #ff00c8;
  --background-dark: #0a0a0f;
}
```
